# DKCR PopTracker (v0.2.0)

This is the PopTracker package designed to work with the ***Donkey Kong Country Returns*** [**APWorld v0.0.2**](https://github.com/CallmeZewo/DKCR_APWorld), built for either manual tracking or automatic tracking through Archipelago. The logic in this pack is based on the original 2010 release of the game on the Nintendo Wii, so if used for any other version of the game, it may have logic inaccuracies. This version of the pack was built for the v0.35.3 release of PopTracker. (It should still work with later versions.) You can install the PopTracker program [here](https://github.com/black-sliver/PopTracker/releases/latest).



### Installation

Download and place the folder containing all of the pack files (named "dkcr\_poptracker\_v0.2.0") into the "packs" folder of your PopTracker installation. It may be either a zipped or unzipped folder, but make sure there aren't two layers of folders to get to the core files. If you have any issues, ping @magicmason1000 in the Archipelago Discord server.



### Important Notes!

* As the APWorld is still in early development, this pack does not have much item functionality yet. Most of the items on the list are all static images, but as more is added into the APWorld, their functionality will be added in on the tracker. Currently, the only items tracked are Puzzle Pieces, KONG Letters, and Balloon filler items.
* Keep in mind that you do not actually receive any items in the game itself yet as there is no patching, so this tracker only tells you what you have on the server's side. This means that for the Boss Puzzle Piece Requirements option, you might be able to enter the boss stage in-game, but according to logic you shouldn't be able to. To stay within logic, only go for checks that are listed green on the tracker, even if your game lets you!
* Map Keys must be manually tracked, as they currently are not recognized by the server.
* This is the first PopTracker pack I've made, so there's bound to be mistakes. Please ping me if anything goes wrong!
